# 🪙 Gold Price Tracker Website

A website that provides **real-time gold prices** along with tool to help users track their gold assets and monitor profit or loss.

## 📌 Features

### 📊 Live Gold Prices

- Displays **live gold prices** with real-time updates.
- Covers multiple gold types:
  - 24K Gold
  - 21K Gold
  - 18K Gold
  - English Coin
  - Rashadi Coin

---

### 💱 Currency Switching

- Easily switch between:
  - **USD ($)**
  - **Jordanian Dinar (JOD)**

- All prices update dynamically based on the selected currency.

---

### 💼 Asset Tracking

- Add your personal gold assets:
  - Enter type, weight, and purchase price

- Automatically calculates:
  - Current value
  - Profit or loss

- Helps you understand how your investments are performing in real-time.

---

## 🛠️ Technologies Used

- HTML, CSS, JavaScript
- APIs for live gold price data
- Local Storage for saving user data
