##  Gold Price Tracker Web App

Welcome to the Gold Price Tracker! This web application allows users to monitor real-time gold market prices, create a secure account, and manage a personal portfolio of gold assets to track their live profit and loss margins.

##  Key Features & Workflow
Built over a 5-day Agile sprint, our modular application includes:
* **Live API Integration:** Fetches real-time global gold prices and dynamically calculates gram/karat values.
* **Secure Authentication:** Custom login and registration system protecting private routes using session storage.
* **Portfolio Manager:** Interactive dashboard for users to save assets, upload images, and calculate dynamic profit/loss margins.
* **Responsive UI:** Built with HTML5, CSS3, Bootstrap 5, and dynamic JavaScript routing.
